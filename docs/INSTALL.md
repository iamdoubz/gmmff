# Installation guide

Each operating system and environment is different. Here is a general outline for what you should do.

1. Download the [latest binary release](https://github.com/iamdoubz/gmmff/releases/latest) for your operating system and platform (e.g. Windows amd64, Linux arm64, etc)
2. Extract the binary from the zip file
3. (Linux) Make the binary executable `sudo chmod +x gmmff`
4. Run the program [see CLI document for help](CLI.md)
